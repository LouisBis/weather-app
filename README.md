# weather-app

## Start project

```bash
cd weather-app
touch .env
```

Add a .env file with `VITE_API_KEY="XXX"`

Replace XXX by your Api Key, available on <https://api.openweathermap.org> with a free account.

```bash
npm i
npm run dev
```

## Start testing part

Make sure the app is running

```bash
px cypress open
```
